**YC S26 批次全景：193 家公司，Agent 原生时代正式到来**

Y Combinator 2026 年春季批次（Spring 2026 / P26）的 193 家公司已全部上线。**这是 YC 历史上 Agent 浓度最高的一批——不是「AI 赋能」，而是「Agent 原生」。** 大部分公司的产品描述里直接写着「AI agents for X」，还有大量公司在构建 Agent 运行所需的底层基础设施：沙箱、记忆、认证、支付、可观测性。

B2B 以 119 家（62%）的绝对优势主导。国防和机器人明显上升。医疗健康几乎全是 AI-for-clinicians，而非生物技术。以下是按行业分类的完整名单。

这份名单最有趣的不是哪家公司，而是整体信号。119 家 B2B 公司里，几乎每一家都在做 Agent 相关的事——不是「AI 辅助」，而是「Agent 原生」。这意味着 YC 判断的下一波企业软件浪潮不是「AI 加在现有产品上」，而是「从第一天起就是 Agent 驱动」。<br><br>
另一个值得注意的信号是基础设施层的密度。沙箱（Indexable、Runtime、Ardent、Arga Labs）、记忆（Memory Store、Wato）、通信（primitive、Chert）、浏览器层（StableBrowse）——Agent 正在催生一套全新的技术栈，就像云时代催生了 AWS 生态一样。这些基础设施公司如果跑出来，可能比上层的 Agent 应用公司更有长期价值。<br><br>
国防和机器人的 23 家公司也值得关注。从机器鸟到核能到太空制造，YC 显然在系统性下注「实体世界 + AI」。这跟五年前纯 SaaS 的 YC 批次已经完全不同了。
