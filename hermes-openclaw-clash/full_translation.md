# 完整翻译 — Hermes Agent vs OpenClaw 公开冲突

Peter Steinberger（OpenClaw 创始人）发推说 OpenClaw hype 消退后团队提升了质量、扩大了规模，成立了非营利组织，而"竞争对手是风投资助的且有其他议程"。这是他们最强的一周。

当被问到"为什么选 OpenClaw 不是 Hermes"时，Peter 回复："他们复制了大量功能，但跳过了安全加固，一份安全报告都没发布过。"

Teknium（Hermes 创始人）直接回复 Peter："我们是不是免费住在你脑子里？我们都知道非营利对 OpenAI 的人来说意味着什么——屁都没有。好好享受被那个非营利组织雇你赚的几百万吧——它用风投的钱和议程来付你工资。"

Teknium 后来补充说："他明明在诋毁我们，你说我回应他说他离开 OpenClaw 去了风投资助的非营利是幼稚？滚吧。""我主动提出过和解，他无视了。现在他诋毁我们，好像我们是风投资助的就是腐败了。"

有用户指责 OpenClaw 产品稳定性差。有人批评 Peter 的双标——OpenClaw 被 OpenAI 收购，本质也是风投钱。Jeff Escalante 说这个回复"太难看"，Henry Mascot 质问 Teknium 为什么回到人身攻击。

OpenClaw 被指 39% GitHub 星标造假并被当面质问。

OpenClaw 团队成员 Vincent Koc 控诉："所有的 gaslighting 和谎言只是为了炒作他们的营销……更不用说来自他们的水军账号的死亡威胁和自杀笑话了。"他贴出了 Discord 截图作为证据。

Hesamation 发布截图推文："绝不离开这个应用。Hermes 和 OpenClaw 的冲突升级太快了。"

用户反应：有人支持 Hermes，说它是"史上最佳"；有人两边都用；有人觉得"两个创始人都容不下别人的自我"；有人总结这是"Mac vs Windows 2.0""爸妈又吵架了"。

社区氛围从良性竞争变成了阵营对立。
