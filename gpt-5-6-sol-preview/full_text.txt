# 产品系列
GPT-5.6 系列包含三个层级：
- Sol（旗舰）— 最强能力，处理最复杂前沿任务。输入$5/百万token，输出$30/百万token
- Terra（均衡）— 日常主力工作模型，性能对标 GPT-5.5 但价格砍半。输入$2.5/百万token，输出$15/百万token
- Luna（入门）— 快速低价。输入$1/百万token，输出$6/百万token

新命名体系：数字=世代，Sol/Terra/Luna = 固定能力层级，可独立迭代。

# 核心能力

## Coding: Terminal-Bench 2.1
Sol 在命令行工作流（需规划、迭代、工具协调）上达到新 SOTA。

## Biology: GeneBench v1
Sol 在长周期基因组学和定量生物学分析上以更少 token 超越 GPT-5.5。

## Cybersecurity
- ExploitBench: Sol 只用 Mythos Preview ~1/3 的 output token 即可达到可比成绩
- ExploitGym (UC Berkeley + OpenAI 联合): Sol, Terra, Luna 三款模型随推理力度增强都展示出显著的网络安全能力提升

关键结论：GPT-5.6 Sol 未跨越 Preparedness Framework 的 Cyber Critical 门槛。在 Chromium 和 Firefox 测试中，能发现漏洞和利用原语，但未自主完成全链利用。

两个新特性：
1. max reasoning effort — 给 Sol 最长时间做深度推理
2. ultra mode — 利用 sub-agents 加速复杂工作，超越单 agent 能力

# 安全体系
四层防护：
1. 模型级训练 — 拒绝被禁止的网络安全帮助，含伪装意图或越狱尝试
2. 实时输出分类器 — 生成过程中检测，高风险时暂停，由更大模型审查
3. 账户级审查 — 跨对话关联风险信号，区分持续恶意行为和合法双用途工作
4. 差异化访问 — 保护防御性工作，默认不开放最敏感能力

# 自动化红队
- 超 70 万 A100 等效 GPU 小时用于自动化红队测试
- 核心目标：寻找通用越狱（跨提示词/跨场景的通用攻击方式）
- 辅以人工专家红队测试
- 快速响应流程：复现→评估→优先级排序→修复→加入基准测试集

# 定价与部署
- 可预测的 prompt caching: 显式缓存断点，30 分钟最小缓存生命期
  - 缓存写入：1.25× 非缓存输入定价
  - 缓存读取：90% 输入费用折扣
- Cerebras 部署：2026 年 7 月，Sol 在 Cerebras 上可达 750 tokens/s
- 有限预览 → 未来几周全面可用
